一等函数

即将函数用作数据值的能力——传递函数为参数、返回函数以及将函数复制给变量和对象属性。


const fibonacci = [1, 1, 2, 3, 5, 8, 13, 21];
const double = x => x * 2;
console.info(fibonacci.map(double));

匿名函数和简洁的 lambda 语法

x => x * 2 在 JavaScript 中是有效的函数表达式。简洁的 Lambda 让它更容易与高阶函数配合。


const fibonacci = [1, 1, 2, 3, 5, 8, 13, 21];
const double = x => x * 2;
console.info(fibonacci.map(double));

闭包

闭包是函数与其词法环境捆绑在一起。闭包是在函数创建时创建的。当一个函数被定义在另一个函数内部时，它可以访问外层函数中绑定的变量，即使在外层函数退出之后也是如此。


const closure = (function () {
   let _name = 'John';
   return {
       getName() {
          return _name;
       }
   };
}());

高阶函数

高阶函数是以一个函数为实参，或者返回一个函数的函数。高级函数与一阶函数相反，一阶函数不能以一个函数为实参或者返回一个函数为输出的。

Array.prototype 的 .map() 和 .filter() 的例子都以一个函数作为实参。二者都是高阶函数。


const result = [1,2,3,4,5,6,7,8,9].filter(x => x > 4).map(x => x * 2);
console.info(result);

Reduce

Reduce 以一个 reducer 函数和一个初始值为参数，并返回一个累加值。


array.reduce(
  reducer: (accumulator: Any, current: Any) => Any,
  initialValue: Any
) => accumulator: Any

[1, 2, 3, 4, 5].reduce((acc, n) => acc + n, 0); // 15

[1, 2, 3, 4, 5].reduce((acc, n) => {
    console.info(`acc=${acc}, n=${n}`);
    return acc + n;
}, 0); // 15

this


function WhoAmI() {
    console.info('I\'m ' + this.name + ' of ' + typeof(this));
}
WhoAmI();

var BillGates = {name: 'Bill Gates', WhoAmI};
BillGates.WhoAmI();

var SteveJobs = {name: 'Steve Jobs', WhoAmI};
SteveJobs.WhoAmI();

WhoAmI.call(BillGates);
WhoAmI.call(SteveJobs);

BillGates.WhoAmI.call(SteveJobs);
SteveJobs.WhoAmI.call(BillGates);

WhoAmI.WhoAmI = WhoAmI;
WhoAmI.name = 'WhoAmI';
WhoAmI.WhoAmI();

({name: 'nobody', WhoAmI}).WhoAmI();

原型继承


function Person(name) {
    this.name = name;
}
Person.prototype.sayHello = function () {
    console.info('Hello, I\'m ' + this.name);
};

function Employee(name, salary) {
    Person.call(this, name);
    this.salary = salary;
}
Employee.prototype = new Person();
Employee.prototype.showMeTheMoney = function() {
    console.info(this.name + ' $ ' + this.salary);
};

var BillGates = new Person('Bill Gates');
BillGates.sayHello();

var SteveJobs = new Employee('Steve Jobs', 1234);
SteveJobs.sayHello();
SteveJobs.showMeTheMoney();

ES6 Class


class Person {
    constructor(name) {
        this.name = name;
    }
    sayHello() {
        console.info('Hello, I\'m ' + this.name);
    }
}
class Employee extends Person {
    constructor(name, salary) {
        super(name);
        this.salary = salary;
    }
    showMeTheMoney() {
        console.info(this.name + ' $ ' + this.salary);
    }
}
var BillGates = new Person('Bill Gates');
BillGates.sayHello();

var SteveJobs = new Employee('Steve Jobs', 1234);
SteveJobs.sayHello();
SteveJobs.showMeTheMoney();
